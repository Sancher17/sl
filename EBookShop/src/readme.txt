
Задание 1 (сложность 2)
Добавить к программе, получившейся в предыдущем задании и всем ее модулям качественную обработку исключительных ситуаций. Все серьезные ошибки программы должны логироваться в файл. Для логирования можно использовать готовое решение, либо реализованное самостоятельно.

Задание 2 (сложность 10)
Добавить к программе из предыдущего задания модуль пользовательского интерфейса (UI). Модуль должен содержать пользовательский консольный интерфейс и позволять полноценно выполнять все функции заложенные в программу. Сделать рефакторинг всего кода программы для приведения его в соответствие принципам MVC.
Требование к задаче:
1)	К модулю UI должна быть разработана детальная диаграмма классов ДО написания кода модуля
2)	Полученная на выходе программа должна соответствовать принципам ООП и шаблона MVC
3)	В модуле UI должны быть использованы следующие паттерны проектирования:
a.	Observer
b.	Singleton
c.	Abstract factory
4)	Модуль пользовательского интерфейса должен быть в отдельном jar файле
