***Меню Order***
0 - главное меню                                                - ОК
1 - добавить заказ
    1.1 если ввел буквы  - сообщение                            - ок
    1.2 если нет такой позициии - сообщение,                    - ок
    1.3 ввел цифра что есть                                     - ОК
2 - удалить заказ
    2.1 если есть позиция                                       - ок
    2.2 если нет такой позициии - сообщение,                    - ок
    2.3 если ввел буквы  - сообщение                            - ок
3 - вывести на экран все заказы                                 - ОК
4 - вывести на экран все выполненые заказы                      - OK
5 - вывести на экран все выполненые заказы / сортировка по дате - OK
6 - вывести на экран все выполненые заказы / сортировка по цене - OK
7 - вывести на экран сумму всех заказов за период               - OK
8 - вывести на экран количество всех выполненых заказов         - OK
9 - вывести на экран заказ                                      - OK
10 - сортировать выполненые заказы по дате                      - OK
11 - сортировать заказы по текущему состоянию                   - OK
12 - сортировать заказы по цене                                 - OK
13 - отметить заказы выполненым                                 - OK
99 - завершение работы                                           -OK

6 - сортировать запросы по количеству                            -OK
